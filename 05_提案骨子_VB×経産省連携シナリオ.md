# 提案骨子：バリューブックス × 経産省「書店振興プロジェクト」連携シナリオ

## 🎯 提案の一言

> **「バリューブックスは、経産省が求める"書店活性化"の解決策を、すでに自ら作り、動かしている民間企業です」**

---

## 📋 提案ストーリーライン

### 導入：「なぜ日本だけが？」

```
世界16カ国中12カ国で書籍売上が増加している
アメリカでは年間323店の新規書店が開業している
ドイツは日本と同じ再販制度なのに書店は成長している
韓国は日本と同じ委託販売なのに書店が復活している

なのに日本だけが、年614店のペースで書店が消え続けている
```

→ **「日本の書店衰退は"宿命"ではない。"流通構造"の問題だ」**

---

### 問題提起：経産省が見抜いた本質

```
経産省が2024年に公表した34項目の課題が示すように、
書店の衰退は「本が売れない」のではなく、
「売っても儲からない構造」が原因である。

粗利22-23% → 人件費・賃料の高騰に耐えられない
取次依存 → 書店が自分で選書・仕入れできない
DX化の遅れ → Amazonに対抗する術がない
```

→ **「構造を変えなければ、いくら支援しても効果は限定的」**

---

### 転換：海外の成功事例が示す「答え」

| 国 | 成功の鍵 | 共通点 |
|----|---------|--------|
| 🇺🇸 アメリカ | イングラムが出版のOSに | **卸がインフラを提供** |
| 🇩🇪 ドイツ | リブリが書店のバックオフィスに | **卸がIT支援** |
| 🇰🇷 韓国 | 教保文庫がOMOで復活 | **オンラインとオフラインの融合** |
| 🇬🇧 イギリス | ドーントが分権型運営で黒字化 | **現場への権限委譲** |
| 🇫🇷 フランス | 国が2,010億円を投じて保護 | **公的支援の本気度** |

→ **「共通する答えは、"書店を支える新しいインフラ"の存在」**

---

### 核心：バリューブックスが、その「インフラ」を作っている

> **私たちは海外の成功モデルを研究し、日本の文脈で統合した「第四のモデル」を、すでに構築し始めています**

#### VBが実現する3つの柱

| 柱 | 内容 | 対応する海外モデル |
|----|------|------------------|
| **① 物流DX基盤** | SKYPOD倉庫で処理能力3倍、翌日配送 | 🇩🇪 リブリ ＋ 🇺🇸 イングラム |
| **② エコシステム** | プロアカウントで書店・著者・出版社を接続 | 🇺🇸 Bookshop.org ＋ 🇰🇷 教保文庫 |
| **③ 粗利革命** | 古本×新刊ハイブリッドで粗利40%超 | 🇺🇸 Powell's ＋ 🇰🇷 アラジン |

## 📊 具体的な提案内容

### 提案A：「書店粗利改善モデル事業」の実証パートナー

#### 概要

経産省の令和6年度補正予算による「書店のDX化・効率化」実証事業に、VBがパートナーとして参画。

#### 実証内容

| フェーズ | 内容 | 期間 |
|---------|------|------|
| **Phase 1** | 独立系書店3〜5店にVBから古本を卸売し、粗利改善効果を検証 | 3ヶ月 |
| **Phase 2** | プロアカウント経由でEC機能を提供し、オンライン売上を検証 | 3ヶ月 |
| **Phase 3** | 結果をデータ化し、全国展開モデルとして公表 | 3ヶ月 |

#### 期待される成果

| 指標 | 現状 | 目標 |
|------|------|------|
| 書店の粗利率 | 22-23% | **35%以上** |
| オンライン売上比率 | ほぼ0% | **15-20%** |
| 返品率 | 高水準 | **大幅削減** |
| 書店の月次利益 | 赤字〜トントン | **黒字化** |

---

### 提案B：「無書店自治体への書店開設支援」

#### 概要

27.9%の無書店自治体に対し、VBのプロアカウント＋低在庫モデルで、**低リスクで書店を開ける仕組み**を提供。

#### モデル

```
【従来の書店開業】
  初期投資: 数千万円
  在庫リスク: 大
  仕入れルート: 取次との契約が必要
  → ハードルが極めて高い

【VBモデルの書店開業】
  初期投資: 数百万円（店舗のみ）
  在庫リスク: ほぼゼロ（VBの倉庫から直送）
  仕入れルート: VBから古本＋新刊を買い切り
  EC機能: VBが提供
  → 誰でも書店が始められる
```

#### 韓国の「地域書店活性化条項」との連動

韓国では自治体が法的根拠を持って書店を支援している。日本でも経産省の書店活性化プランと自治体の連携が進めば、VBのモデルが「全国の無書店地域に書店を作る」インフラになる。

---

### 提案C：「出版流通改革の実証プラットフォーム」

#### 概要

日販・トーハンの取次システムとは別の「第三の流通ルート」として、VBが中小出版社と独立系書店をつなぐ実証プラットフォームを構築。

#### 関係図

```
【現在の流通】
出版社 → 取次（日販・トーハン）→ 書店
          粗利22-23%
          自動配本
          返品可

【VBが提案する第三の流通】
出版社 → VB（埼玉倉庫）→ 書店
   ↑        ↓
   ├── 古本在庫で品切れ補完
   ├── PODで絶版本復活
   ├── データで仕入れ最適化
   └── EC機能＋受け取り拠点
          粗利35-40%超
          書店が選んで仕入れ
          買い切り（返品なし）
```

#### ブックセラーズ＆カンパニーとの棲み分け

| 項目 | ブックセラーズ＆カンパニー | バリューブックス |
|-----|------------------------|-----------------|
| 出資者 | 紀伊國屋・CCC・日販 | 独立企業 |
| ターゲット | 大手書店チェーン | **独立系書店・中小出版社** |
| モデル | 新刊中心の直取引 | **新刊×古本ハイブリッド** |
| 方向性 | 取次改革 | **取次を介さない第三の道** |

> **VBは「大手が手を出さない領域」＝経産省が最も支援したい独立系書店をカバー**

## 💬 経産省への伝え方

### キーメッセージ1：「再販制度は言い訳にならない」

```
ドイツには日本と同じ再販制度がありますが、書店は成長しています。
違いは「流通構造」です。
バリューブックスは、ドイツ型の「書店のバックオフィス」を日本で作ります。
```

### キーメッセージ2：「委託販売でも復活できる」

```
韓国は日本と同じ委託販売ですが、書店は復活しています。
韓国の成功は「公的支援×OMO」の組み合わせです。
バリューブックスは「OMOインフラ」を自ら作り、
経産省の「公的支援」と組み合わせれば、韓国以上のモデルが実現します。
```

### キーメッセージ3：「粗利を上げれば書店は生き残れる」

```
書店の粗利は22-23%。これでは経営は成り立ちません。
バリューブックスから古本を30%混ぜるだけで、粗利は35%に。
50%混ぜれば40%超に。
これは「理論」ではなく、すでに動いている「仕組み」です。
```

### キーメッセージ4：「日本にも"イングラム"が必要」

```
アメリカの独立系書店復活を支えたのは「イングラム」という
出版インフラの存在です。
日本にはそれがありません。
バリューブックスは「日本版イングラム」を目指しています。
SKYPOD倉庫、プロアカウント、古本×新刊ハイブリッド。
すべてはこの目標のために動いています。
```

### キーメッセージ5：「B Corpの信頼」

```
バリューブックスはB Corp認証を取得した
「社会に良い企業」です。
書店支援は「ビジネス」であると同時に「社会貢献」です。
charibonで実績があり、メディア（積読チャンネル等）も持っています。
```

## 📐 提案の強み・弱みの自己分析

### 強み

| 項目 | 内容 |
|-----|------|
| **実行力** | すでに動いている事業がある（構想ではない） |
| **データ** | 日本最大級の古本市場データを保有 |
| **インフラ** | SKYPOD倉庫、物流網が進行中 |
| **ブランド** | B Corp認証、メディア力 |
| **研究** | 5カ国の成功モデルを徹底研究 |
| **ポジション** | 独立系書店の味方（大手とバッティングしない） |

### 弱み・リスク

| 項目 | 内容 | 対策 |
|-----|------|------|
| **規模** | 大手取次と比べれば小さい | まず小さく成功事例を作る |
| **新刊流通の実績** | 古本が主力、新刊は発展途上 | 実証事業で実績を積む |
| **書店との関係** | 独立系書店との取引実績はこれから | プロアカウントで信頼構築 |
| **政府との関係** | 経産省とのパイプはこれから | 提案と実績で信頼を勝ち取る |
