# 海外事例サマリー：5カ国の成功モデルとVBへの示唆

## 📊 各国比較一覧

| 国 | 流通モデル | 再販制度 | 書店動向 | 粗利率 | キーワード | VBとの接続 |
|----|----------|---------|---------|-------|-----------|-----------|
| 🇺🇸 アメリカ | 直取引・買い切り | なし | **+323店/年** | 40%〜 | イングラム、POD、ドーント | 日本版イングラム |
| 🇩🇪 ドイツ | 買い切り＋IT支援 | **あり** | **+3.1%成長** | 高い | リブリ、翌朝配送 | 書店のバックオフィス |
| 🇰🇷 韓国 | 委託販売 | なし | **拡大中** | - | 教保文庫、OMO、公的支援 | 受け取り拠点、公的連携 |
| 🇬🇧 イギリス | 買い切り | なし（1995廃止） | **復活中** | 35-45% | ドーント、チャリティ | 古本×社会貢献 |
| 🇫🇷 フランス | 買い切り | **あり** | **+1.5%成長** | - | 反アマゾン法、公的支援 | 公的支援のモデル |
| 🇯🇵 **日本** | 委託販売 | **あり** | **-614店/年** | 22-23% | 取次依存、雑誌崩壊 | **VBが変える** |

> **再販制度は言い訳にならない**。ドイツもフランスも再販制度があるが書店は成長している。

## 🇺🇸 アメリカ：「日本版イングラム」の原型

### 核心：イングラムが出版インフラのOSになった

| イングラムの機能 | VBでの対応 |
|----------------|-----------|
| **仮想在庫（Virtual Inventory）** | 古本在庫で「品切れゼロ」を実現 |
| **POD（オンデマンド印刷）** | 埼玉倉庫にPOD機導入構想 |
| **IngramSpark（誰でも出版）** | プロアカウントで著者・出版社が直販 |
| **IPS（営業代行）** | VBのプラットフォームが書店への販路 |
| **Global Connect（国際展開）** | データだけが国境を越えるモデル |

### 独立系書店復活のデータ

| 指標 | 数値 |
|-----|------|
| 2024年新規開業 | **323店舗（+31%）** |
| 2024年閉店 | 37店舗（-61%） |
| ABA会員数増加（5年間） | **+1,000店以上** |

### ドーントの革命（B&N再建）

> 「皮肉なことに、責任を店舗ごとに下ろすと、彼らははるかに上手にやってくれるのだ」

| 改革 | 内容 | VBへの教訓 |
|-----|------|-----------|
| 中央集権の廃止 | 店舗ごとに選書権限を委譲 | 独立系書店の自律性を尊重 |
| 有料棚の廃止 | 出版社の「棚買い」を撤廃 | データによる選書支援 |
| 返品率の低下 | 現場主導で売れ残り激減 | 買い切り＋レコメンド |

### 注目モデル：新刊×古本ハイブリッド

| 書店 | 特徴 | VBとの類似性 |
|-----|------|------------|
| **Powell's Books** | 新刊と古本を同じ棚に | **VBのハイブリッド卸そのもの** |
| **Troubled Sleeper** | 新刊×古本の新興モデル | VB構想の正当性を証明 |
| **McNally Jackson** | 店内にPOD機設置 | 埼玉倉庫POD構想 |

### 経産省への訴求ポイント

> **アメリカで年間323店の新規開業を支えているのは、イングラムという「書店のインフラ」の存在。VBはその日本版を目指している。**

## 🇩🇪 ドイツ：「書店のバックオフィス」モデル

### 核心：再販制度があっても成長している唯一のモデル

**日本との共通点**：再販制度あり（定価販売）
**日本との違い**：流通が「買い切り＋IT支援」

| ドイツの仕組み | 日本の現状 | VBが実現すること |
|-------------|---------|----------------|
| 翌朝配送が標準 | 数日かかることも | 埼玉倉庫から翌日配送 |
| 卸がITシステムを提供 | 各社バラバラ | VBが統合プラットフォーム提供 |
| 書店が自由に選書 | 取次の配本に依存 | プロアカウントで書店が選ぶ |
| Amazon同等の利便性 | Amazonに劣る | VB提携書店でAmazon対抗 |

### リブリ（Libri）のサービス

| サービス | 内容 | VBの対応 |
|---------|------|---------|
| オーバーナイト・デリバリー | 夕方注文→翌朝到着 | 埼玉倉庫からの高速配送 |
| 在庫管理システム | 書店の在庫を卸が管理 | SKYPOD＋在庫管理ツール |
| ECサイト構築 | 独立系書店にEC機能提供 | プロアカウントのEC機能 |
| 在庫分析 | 売れ筋データ提供 | VBの買取データ活用 |
| 決済代行 | 出版社との精算代行 | VBプラットフォームの決済 |

### 経産省への訴求ポイント

> **「再販制度があるから書店は衰退する」は誤り。ドイツは同じ制度で成長している。違いは"卸の役割"。VBはドイツ型の「書店のバックオフィス」を日本で実現する。**

## 🇰🇷 韓国：「公的支援」×「OMO」の復活モデル

### 核心：日本と同じ委託販売でも復活できた

**日本との共通点**：委託販売制度
**日本との違い**：手厚い公的支援＋OMO戦略

### 教保文庫の「バロバロ」サービス

| 項目 | 内容 | VBでの対応 |
|-----|------|-----------|
| オンライン注文 | ネットで本を注文 | VBのECサイト |
| 1時間受け取り | 最寄り店舗で即受取 | 提携書店での受け取り（Phase 1は翌日） |
| 店舗回遊 | 受取ついでに追加購入 | 書店への送客効果 |

### 韓国の公的支援制度

| 施策 | 内容 | VBとの接続 |
|-----|------|-----------|
| 出版文化産業振興法 | 法律で書店を支援 | 経産省の書店活性化プランと同方向 |
| 地域書店活性化条項 | 自治体の支援に法的根拠 | VBが自治体と連携する根拠 |
| 図書館×書店連携 | 市立図書館が地元書店から購入 | VBが図書館向け卸にもなれる |
| ブックフェア支援 | 市が経費を負担 | VBの「スローブック運動」と連動 |

### アラジン：韓国版ハイブリッドモデル

| 特徴 | 内容 | VBとの類似性 |
|-----|------|------------|
| 新刊販売 | 通常の書店機能 | VBの新刊販売戦略 |
| 中古買取 | 古本の買取サービス | VBの本業 |
| 中古販売 | 古本の店頭販売 | VBの在庫活用 |
| オンライン連携 | EC販売も充実 | VBのEC基盤 |

### 経産省への訴求ポイント

> **韓国は日本と同じ委託販売でも復活した。公的支援＋OMO戦略がカギ。VBは「公的支援を待つ」のではなく「自らOMOインフラを作る」ことで先行している。経産省と連携すれば韓国以上のモデルが作れる。**

## 🇬🇧 イギリス：「古本×社会貢献」の成熟モデル

### 核心：チャリティショップ文化 ＝ charibon（チャリボン）の完成形

### チャリティショップの仕組み

```
市民が不要な本を寄付
  ↓
チャリティが店頭で販売
  ↓
売上が社会貢献活動に充当
  ↓
購入者に「社会に良いことをした」満足感
```

> **バリューブックスの「charibon（チャリボン）」と同じ発想が、イギリスでは巨大産業になっている**

### World of Books（大規模オンライン古本販売）

| 特徴 | 内容 | VBとの比較 |
|-----|------|-----------|
| 在庫数 | 数百万冊 | VBも日本最大級 |
| 調達方法 | チャリティ提携＋買取 | charibon＋買取 |
| 環境訴求 | 「本を救う」 | VBのB Corp認証 |
| 販売チャネル | Amazon+eBay+自社 | マルチチャネル展開 |

### ドーントの原点：ウォーターストーンズ再建

| 2011年 | 現在 |
|--------|------|
| 赤字（数千万ポンド） | 黒字 |
| 返品率が高い | 大幅低下 |
| Amazon敗北 | 共存（棲み分け） |

### 経産省への訴求ポイント

> **VBのcharibon（チャリボン）は、イギリスで大成功しているチャリティ×古本モデルの日本版。B Corp認証と合わせて「社会的意義のある書店インフラ」として経産省にアピールできる。**

## 🇫🇷 フランス：「公的支援の最高峰」

### 核心：書籍を「生活必需品」として国が全力で支援

| 施策 | 内容 | 金額規模 |
|-----|------|---------|
| VAT軽減税率 | 20%→5.5% | 税収減1,090億円 |
| 反アマゾン法 | 無料配送禁止 | - |
| カルチャー・パス | 18歳に300ユーロ支給 | - |
| SOFIA制度 | 図書館購入費の6%を還元 | - |
| **年間支援総額** | - | **約2,010億円** |

### 経産省への訴求ポイント

> **フランスは年間2,010億円を書店支援に投じている。日本は経産省の補正予算が第一歩。VBのような「民間で動いている解決策」と連携することで、少ない予算でもフランス型の効果を出せる。**

## 📊 5カ国の成功要素とVBの統合

### VBが目指す「第四のモデル」

| 取り入れる要素 | 出典国 | VBでの実装 |
|--------------|--------|-----------|
| POD・仮想在庫 | 🇺🇸 アメリカ（イングラム） | 埼玉倉庫＋古本在庫 |
| 分権型運営・現場権限 | 🇺🇸 アメリカ（ドーント） | 独立系書店の自律性尊重 |
| 高速物流・IT支援 | 🇩🇪 ドイツ（リブリ） | SKYPOD＋統合プラットフォーム |
| OMO・受け取り拠点 | 🇰🇷 韓国（教保文庫） | 提携書店での受け取り |
| 公的支援との連携 | 🇰🇷🇫🇷 韓国・フランス | 経産省書店活性化プラン |
| 古本×社会貢献 | 🇬🇧 イギリス（チャリティ） | charibon＋B Corp |
| 古本×新刊ハイブリッド | 🇯🇵 日本オリジナル | **VB独自の強み** |

> **世界のどこにもない「第四のモデル」＝ VBの独自構想**

## 💡 経産省向けメッセージ

### 「世界中で書店は復活している。日本だけが取り残されている」

| 世界のトレンド | 日本の現状 |
|-------------|---------|
| 世界16カ国中12カ国で書籍売上増加 | 30年間右肩下がり |
| アメリカは年+323店の新規開業 | 年-614店の閉店 |
| ドイツは再販制度下で+3.1%成長 | 再販制度下で衰退 |
| 韓国は委託販売でも復活 | 委託販売のまま衰退 |

### 「なぜ日本だけが？」→ 「流通構造」の問題

1. 取次による中央集権的な配本システム
2. 書店の粗利が22-23%と低すぎる
3. 書店にDX・EC・データ活用のインフラがない

### 「誰が解決するのか？」→ バリューブックス

1. **すでに日本最大級の古本在庫**を持つ
2. **SKYPOD倉庫**で物流DXが進行中
3. **プロアカウント**で書店・著者・出版社のエコシステムを構築中
4. **B Corp認証**で社会的信頼がある
5. **海外の成功モデルを研究し尽くしている**
