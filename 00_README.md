## 📊 提案のロジック構造

```
【経産省の課題感】
  書店が年614店閉店、27.9%の自治体が書店ゼロ
  → 文化創造基盤の崩壊、コンテンツ産業の危機
     ↓
【経産省が求めるもの】
  ① 書店の粗利改善（22%→30%以上）
  ② 流通・物流の効率化（DX化）
  ③ 地域書店の持続可能なモデル
  ④ 海外成功事例の日本版実装
     ↓
【バリューブックスが提供できるもの】
  ① 古本×新刊ハイブリッドで粗利40%超を実現
  ② SKYPOD倉庫＋IT支援で「書店のバックオフィス」に
  ③ プロアカウントで著者・書店・出版社をエコシステム化
  ④ 米独韓英仏の成功モデルを統合した「第四のモデル」
     ↓
【提案】
  VBの事業は「書店活性化プラン」の実証パートナーとして最適
```

## 💡 経産省が注目する海外事例とVBの接続

| 海外事例 | 経産省の関心 | VBの対応 |
|---------|------------|---------|
| 🇩🇪 ドイツ「リブリ」 | 卸がITインフラを提供、翌朝配送 | **埼玉倉庫が「日本版リブリ」に** |
| 🇺🇸 アメリカ「イングラム」 | POD・仮想在庫で在庫リスクゼロ | **SKYPOD＋PODで日本版イングラム** |
| 🇰🇷 韓国「教保文庫」 | OMO戦略、受け取り拠点 | **プロアカウントで書店をEC拠点化** |
| 🇫🇷 フランス | 公的支援＋流通改革 | **B Corp×社会的事業としての訴求** |
| 🇬🇧 イギリス「チャリティショップ」 | 古本×社会貢献モデル | **charibon（チャリボン）の実績** |
