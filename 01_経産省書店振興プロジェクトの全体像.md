# 経産省「書店振興プロジェクト」の全体像

## 📊 プロジェクト概要

### 発足の背景

| 項目 | 内容 |
|-----|------|
| **発足時期** | 2024年3月 |
| **体制** | 経済産業大臣直轄「書店振興プロジェクトチーム」 |
| **推進体制** | 経産省＋文科省＋文化庁＋公正取引委員会＋総務省 他 |
| **きっかけ** | 齋藤健経産大臣（当時）が書店危機を問題提起 |

### 危機的状況

| 指標 | 数値 |
|-----|------|
| **現在の書店数** | 10,918店（10年前から3割以上減少） |
| **年間閉店数** | 614店舗（1日1軒以上） |
| **年間新規開店数** | 92店舗 |
| **無書店自治体** | 27.9%（全国の約4分の1） |
| **紙出版市場**（2023年） | 1兆612億円（ピーク時の40%） |
| **月1冊も読まない人** | 62.6%（初めて6割超え） |

> **日本だけが「30年間右肩下がり」という世界的に見ても特異な現象**

## 🏛️ 経産省が書店支援に乗り出す理由

### 単なる商業支援ではない

経産省は書店を**「文化創造基盤」**と位置づけている。

| 位置づけ | 説明 |
|---------|------|
| **文化創造の基盤** | 多様な価値観に触れ、創造性を育む場 |
| **コンテンツ産業の源泉** | マンガ・アニメ・小説など日本のソフトパワーの流通拠点 |
| **国力・競争力** | コンテンツ産業は国家の競争力に直結 |
| **一覧性の価値** | アルゴリズムでは得られない「偶然の出会い」の場 |
| **地域文化拠点** | 地域の知的インフラとしての役割 |

### 経産省の名言

> 「書店がなくなるということは、日本のソフトパワーの源泉が枯れるということ」

## 📋 関係者から指摘された34項目の課題（2024年10月公表）

### 課題の全体構造

経産省は書店関係者への「車座ヒアリング」を通じて課題を集約し、2024年10月に「関係者から指摘された書店活性化のための課題（案）」として34項目を公表。パブリックコメントも実施。

### 主要カテゴリ

#### 1. 流通・物流の構造問題

| 課題 | 内容 |
|-----|------|
| **委託販売・返品制度** | 返品可能な代わりに粗利が22-23%と低すぎる |
| **配本システム** | 取次による自動配本で、書店の選書権限が弱い |
| **返品率の高さ** | 物流コスト増大、2024年問題で配送維持が困難 |
| **取次依存** | 日販・トーハンへの依存度が高く、柔軟な対応ができない |

#### 2. コスト・収益構造

| 課題 | 内容 |
|-----|------|
| **粗利率の低さ** | 22-23%では経営が成立しない（必要水準は30%以上） |
| **人件費高騰** | 最低賃金が800円→1,500〜1,700円に急騰 |
| **賃料高騰** | 商業施設の優遇消滅、不動産高騰 |
| **光熱費・相続税** | 固定費の増加が止まらない |

#### 3. DX化・デジタル対応

| 課題 | 内容 |
|-----|------|
| **キャッシュレス化** | 対応が遅れている書店が多い |
| **EC対応** | ネット書店に対抗できるオンライン機能がない |
| **在庫管理** | アナログな管理が非効率を生んでいる |
| **データ活用** | 販売データの分析・活用が不十分 |

#### 4. 人材・雇用

| 課題 | 内容 |
|-----|------|
| **人手不足** | 低賃金で人材確保が困難 |
| **後継者不在** | 老舗書店の事業承継問題 |
| **書店員のスキル** | DX対応のリテラシー不足 |

#### 5. 競争環境

| 課題 | 内容 |
|-----|------|
| **Amazon等との競争** | 価格・配送スピードで劣る |
| **電子書籍の台頭** | 紙の本の需要自体が減少 |
| **雑誌売上の激減** | かつての収益の柱が崩壊 |

## 📑 「書店活性化プラン」（2025年6月公表）

### プランの方向性

パブリックコメント結果を反映し、関係省庁の施策を統合したプランを2025年6月に公表。

| 施策分野 | 主な内容 |
|---------|---------|
| **DX化・効率化** | 書店のデジタル化、在庫管理の効率化支援 |
| **返品率改善** | データに基づく配本最適化の実証事業 |
| **新規開業支援** | 書店経営者向け支援施策活用ガイドの提供 |
| **流通改革** | 直取引・買い切り制への移行支援 |
| **地域活性化** | 書店を地域文化拠点として支援 |

### 令和6年度補正予算

| 項目 | 内容 |
|-----|------|
| **対象** | 書店のDX化、作業効率化、返品率改善 |
| **形態** | 実証事業（モデル事業） |
| **目的** | 成功モデルの横展開 |

## 🌍 経産省が参考にしている海外事例

### フランス

| 施策 | 内容 |
|-----|------|
| **軽減税率** | 書籍のVATを20%→5.5%に（税収減1,090億円） |
| **反アマゾン法** | 無料配送を法律で禁止（2014年） |
| **カルチャー・パス** | 18歳の若者に300ユーロ支給（2021年〜） |
| **SOFIA制度** | 図書館の書籍購入費の6%を著者・出版社に分配 |
| **年間支援総額** | 約2,010億円（2024年） |

> フランスは書籍を**「生活必需品」**として扱っている

### 韓国

| 施策 | 内容 |
|-----|------|
| **出版文化産業振興法** | 出版産業を「国家知識競争力」と法的に位置づけ |
| **地域書店活性化条項** | 2021年追加、自治体の書店支援に法的根拠 |
| **出版文化産業振興院** | 専門の政府機関を設置 |
| **地域ブックフェア** | 市が経費負担してイベント開催 |
| **図書館×書店連携** | 市立図書館が地元書店から書籍購入 |

> 韓国は独立系書店が**増加傾向**に転じている

## ⚡ 経産省の課題認識と「求めている解決策」

### 経産省が求めているもの（推定）

| 求めているもの | 具体的に |
|--------------|---------|
| **粗利改善のモデル** | 22-23%の粗利を30%以上に引き上げる具体策 |
| **流通構造の改革案** | 取次依存から脱却する現実的な道筋 |
| **DX化の成功事例** | 書店の在庫管理・EC・データ活用の具体例 |
| **地域活性化モデル** | 書店を地域文化拠点として再構築する方法 |
| **海外事例の日本版** | ドイツ・韓国・フランスの成功を日本で実現する具体策 |
| **実証パートナー** | 補正予算の実証事業を一緒に進められる民間企業 |

### 経産省が特に注目しているキーワード

1. **「一覧性」**：書店でしか得られない偶然の出会い
2. **「コンテンツ産業の基盤」**：ソフトパワーの源泉
3. **「地域文化拠点」**：地方の知的インフラ
4. **「DX化」**：テクノロジーによる効率化
5. **「流通改革」**：委託販売から買い切りへの転換
6. **「返品率改善」**：物流の効率化
